Noose

Simple lasso (noose) tool in vanilla JavaScript. Inspired by jQuery UI's selectable. Supports modern browsers and mobile. Noose is kept simple for performance. It's only purpose is to calculate the region to draw and then determine the elements to mark as selected. Styling is left to the user.

Getting Started

Installation

Download or install from git or npm.

npm -i noose
git clone https://github.com/tokafew420/noose.git

or

Use a CDN (jsDelivr).

<script src="https://cdn.jsdelivr.net/npm/noose/dist/noose.min.js"></script>

Distribution

These files are available in the distribution directory.

  • noose.js - ES6 source file
  • noose.min.js - Minified ES6 source file
  • noose.es5.js - Transpiled ES5 source file
  • noose.es5.min.js - Minified transpiled ES5 source file

Basic Usage

Initialize Noose with default options by passing in a valid css selector or a reference to a Element. 

var noose = new Noose('.my-container');

See example

Options

mode

string - Specify the selection criteria. There are 2 possible values.

  • fit - The candidate element must fit entirely inside the noose region.
  • touch - Any part of the candidate element is touching the noose region.
Default is touch.

See example

select

string - Specify the selector for selecting child elements. Default is *.

This is recommended to narrow the scope of what is selectable.

See example

Setting this value to false will bypass the selection process. Do this if you don't care for selecting elements but want the noose to be drawn.

See example
